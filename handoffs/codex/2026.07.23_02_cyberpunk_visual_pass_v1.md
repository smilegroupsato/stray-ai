## Visual target

The target is stronger than the existing restrained theme, while remaining readable and mobile-safe.

### Shared shell

Create a recognizable terminal enclosure around the primary content:

- strong but thin cyan/magenta outer frame
- clipped or stepped corner treatment using CSS
- restrained inner highlight and low-opacity glow
- dark layered background rather than a flat page
- CSS-only grid, scanline, and low-level noise impression
- body text must remain on clean enough surfaces for long Japanese reading

Prefer pseudo-elements and layered gradients. No image data URI except the existing SVG favicon.

### Title zone

Strengthen the top identity area:

- larger Stray AI mark on wide screens
- more deliberate mark/title relationship
- cyan title glow with restrained magenta secondary edge
- compact terminal-style metadata treatment
- title zone should feel like an identity header, not a generic page heading
- maintain responsive wrapping and prevent horizontal overflow

### Current Board

Preserve the information order and content.

Improve:

- NOW as the strongest yellow-accented focal area
- LIVE as a cyan telemetry strip rather than plain metric cards
- NEXT, HOLD / WAIT, RECENTLY DONE, PARKING LOT, and NOT DOING with coherent but distinct accent identities
- panel depth through layered borders, inset shadows, and controlled glow
- completed-foundation group clearly separated from its child items
- child items should read as nested records or modules, not top-level peers
- links remain ordinary read-only text links, not buttons

Selective asymmetry is allowed through small offsets, varied border accents, or stepped headers. Do not reorder semantic content or create masonry behavior that harms reading order.

### Visit Report archive

Strengthen continuity with Current Board:

- same terminal frame and title identity
- state metrics should resemble recorded telemetry
- Visit cards should feel like archived passages/records
- latest badge and source/navigation links remain readable
- retain all current relative-link behavior

### Individual Visit Report

Strengthen:

- route as a visual traversal line
- terminal/leave nodes with stronger distinction
- Brain decision cards as observation records
- Trace/memory/current-state sections as separate evidence modules
- report remains an observation archive, never a control surface
